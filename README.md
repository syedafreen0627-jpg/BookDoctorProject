# Book Doctor

## Online Doctor Appointment Booking System

### Project Description

Book Doctor is a MERN Stack web application that allows patients to register, log in, search for doctors, and book appointments online. The application also provides an admin dashboard where administrators can manage doctors and appointments.

## Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- Bootstrap
- JWT Authentication

## Features

- User Registration
- User Login
- View Doctors
- Book Appointment
- My Appointments
- Admin Dashboard
- Add Doctor
- Edit Doctor
- Delete Doctor
- Protected Routes
